Algorithm Visualizer Insertion Sort

Insertion sort is a simple sorting algorithm that builds the final sorted array one item at a time. It iterates through the input array, comparing each element with the elements to its left, and inserts it into its correct position in the sorted array. The algorithm is efficient for small data sets but becomes inefficient for larger data sets due to its O(n^2) time complexity in the worst case. However, it has several advantages, including simplicity, adaptive behavior, and stability, making it useful in certain scenarios.
