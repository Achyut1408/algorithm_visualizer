Algorithm Visualizer Linear Search

Linear search, or sequential search, is a basic algorithm used to find a specific element in a collection like an array. It checks each element one by one, starting from the beginning until the target element is found or the end of the collection is reached. If a match is found, the algorithm returns the index or position of the element; otherwise, it signals absence, commonly with a return value like -1. While simple, its time complexity is O(n), making it less efficient for large datasets compared to more advanced algorithms like binary search (O (log n)), especially when dealing with sorted collections.
